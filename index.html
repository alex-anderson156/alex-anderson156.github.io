Projects & Example Work

Analyser Control Suite

This project was born out of the thinning of our development team and increase in workload; we needed a new modern framework that would speed up development of applications and reduce the time it took to train users of the system.

This project was built from the ground up over the period of a year, using WPF as the framework of choice for building the graphical user interface due to my expertise with the MVVM design pattern and the customisation of WPF UI elements.

I decided to build an applications system, similar to that used by Android where each application is built upon a stack of activities, each activity has its own singular responsibility, GUI and memory allowing activity states to persist around the users activity.

We already had solutions using SQL databases so we adapted the Microsoft's Entity Framework to provide a data access layer to our application. This was designed to allow developers to easily and safely manipulate the database ensuring A.C.I.D principles were observed, and database was easily rolled back if a error was detected.

As the project was to house many different data analysers, we required a strong, fast and flexible charting library. The open source solution OxyPlot was chosen. I created an abstraction layer using the builder design pattern to hide any Oxyplot specifics from the developers, this meant that Oxyplot could easily be removed and replaced with minimal impact on other applications.
